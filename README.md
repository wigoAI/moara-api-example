# moara-api-example
- moara-core 를 활용한 기본적인 api example
- 자세한 사항은 wigo-api page 를 참조해주세요
- 이 프로젝트는 정리가 되면 더이상 업데이트 되지않고 wigoai-api-example 프로젝트가 새로 생길 예정 입니다.

# 개발환경
- open jdk 1.8
- moara-core last version
